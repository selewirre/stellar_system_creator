Orbital Stability
=================

.. _orbital_stability:

Orbital stability demonstrates if the orbit of the child object around the parent object
does not exceed any limits. More specifically, for a stable orbit we must have:

1. :ref:`Periapsis <periapsis>` > :ref:`roche limit <roche_limit>`.
2. :ref:`Semi-major axis <semi_major_axis>` > :ref:`p-type critical orbit <p_type_critical_orbit>`.
3. :ref:`Apoapsis <apoapsis>` < :ref:`semi-major axis maximum limit <semi_major_axis_maximum_limit>`.
4. Optional: :ref:`Semi-major axis <semi_major_axis>` > :ref:`parent inner orbit limit <inner_orbit_limit>`.
5. For rock worlds: :ref:`Semi-major axis <semi_major_axis>` > :ref:`parent outer rock formation limit <outer_rock_formation_limit>`.
