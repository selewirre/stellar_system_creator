Tidal Heating
=============

.. _tidal_heating:

When an child object of :ref:`radius <radius>` :math:`r` rotates around a parent object
of :ref:`mass <mass>` :math:`M` in an non-circular (eccentric) orbit, at a
:ref:`mean distance <semi_major_axis>` :math:`a` and :ref:`eccentricity <eccentricity>` :math:`e`,
the shape of the child object can periodically change due to the variation of gravitational pull
it feels from the parent object (it wobbles). This motion results in internal heating.

Tidal heating is proportional to: :math:`Q_{\rm tidal} \propto M^{2.5} r^5 e^2 a^{-7.5}`
