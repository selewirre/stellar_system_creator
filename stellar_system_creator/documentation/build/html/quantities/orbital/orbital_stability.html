Orbital StabilityÔÉÅ

Orbital stability demonstrates if the orbit of the child object around the parent object does not exceed any limits. More specifically, for a stable orbit we must have:

  1. Periapsis > roche limit.

  2. Semi-major axis > p-type critical orbit.

  3. Apoapsis < semi-major axis maximum limit.

  4. Optional: Semi-major axis > parent inner orbit limit.

  5. For rock worlds: Semi-major axis > parent outer rock formation limit.
